3 cách để chèn mã vào trong 1 trang html
            c1: chèn luôn vào trong dòng cần css
            c2: viết trong cùng 1 trang với html
            c3: tạo 1 file CSS riêng < nên dùng cách này >
        
            cách thuộc tính nền trong CSS
            1. backgroud-color ( màu nền )
            2. backgroud-image ( ảnh nền )
            3. backgroud-repeat ( có nên lặp ảnh nền khi mà kích thước lớn hơn ảnh nền )
            4. backgroud-position ( xác định vị trí của ảnh nền )
            5. background-attachment ( ảnh nền có cố định trong 1 file html không )

            Chú ý :
            trong các thuộc tính CSS luôn có giá trị mặc định
            ví dụ thuộc tính background-color ( không màu )
            thuôc tính background-image ( none ) không có ảnh nền
            thuộc tính background-repeat ( repeat ) luôn luôn lặp lại
            Khi giá trị mặc định có viết hay không cũng không có tác dụng


            Cú pháp background-image :
            background-image: url("đường dẫn của hình ảnh");
